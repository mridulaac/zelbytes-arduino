Hardware Details

1. Arduino Uno R3
- Microcontroller: ATmega328P
- Operating Voltage: 5V
- Input Voltage: 7V–12V (recommended)
- Digital I/O Pins: 14
- PWM Pins: 6
- Analog Input Pins: 6
- Flash Memory: 32 KB
- SRAM: 2 KB
- Clock Speed: 16 MHz

2. LED (Light Emitting Diode)
- Type: 5 mm LED
- Operating Voltage: 2V–3V
- Operating Current: 10–20 mA
- Function: Visual indication of ON/OFF state

3. 220 Ω Resistor
- Resistance: 220 Ohms
- Power Rating: 1/4 Watt
- Function: Limits current through the LED to prevent damage

4. USB Cable
- Type: USB A to USB B
- Function:
  - Powers the Arduino
  - Uploads sketches
  - Enables Serial Monitor communication

5. Breadboard (if used)
- Type: Solderless breadboard
- Function: Temporary circuit connections without soldering

6. Jumper Wires
- Type: Male-to-Male
- Function: Connect Arduino pins to circuit components
