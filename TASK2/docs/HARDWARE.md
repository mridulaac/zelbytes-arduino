1. Arduino Uno R3
ATmega328P microcontroller-based development board.
Operates at 5V.
Used to control the LED and handle serial communication.

2. LED (Light Emitting Diode)
Electronic component that emits light when current flows through it.
Used as an output indicator.

3. 220 Ω Resistor
Connected in series with the LED.
Limits current and protects the LED from damage.

4. USB Cable
Used to upload the program to Arduino.
Provides power and serial communication with the computer.

5. Breadboard
Used for making temporary circuit connections without soldering.

6. Jumper Wires
Used to connect the Arduino and external components.
