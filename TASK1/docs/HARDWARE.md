ARDUINO UNO

The Arduino Uno is the main controller board used in this project. It is based on the ATmega328P microcontroller and is widely used for embedded system and IoT applications due to its simplicity and ease of programming.

SPECIFICATIONS

* Microcontroller: ATmega328P
* Operating Voltage: 5 V
* Input Voltage (Recommended): 7–12 V
* Input Voltage (Limit): 6–20 V
* Clock Speed: 16 MHz
* Digital I/O Pins: 14
* PWM Pins: 6
* Analog Input Pins: 6
* Flash Memory: 32 KB
* SRAM: 2 KB
* EEPROM: 1 KB
* USB Interface: USB Type-B
* Communication Protocols: UART, I²C, SPI
* Built-in LED: Pin 13

FEATURES

* Easy to program using the Arduino IDE.
* Supports a wide range of sensors and modules.
* Provides both digital and analog input/output capabilities.
* Includes built-in communication interfaces for device connectivity.
* Can be powered through USB or an external power source.

FUNCTION

* Acts as the central processing unit of the system.
* Controls the LED and relay module according to the uploaded program.
* Processes input signals and generates appropriate output signals.
* Enables communication between hardware components and the control logic.
